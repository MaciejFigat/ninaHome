
## Installation
1. cd frontend
2. npm install
3. npm start

## Structure
```
├── 
├── frontend
|   ├── node_modules
|   ├── public
│   │   ├── favicon.ico
│   │   ├── index.html
│   ├── src
│   │   ├── assets (photos, graphics etc.)
│   │   ├── components (reusable components) 
|   |   │   ├── animated - (contains framer-motion components)
│   │   ├── data - files containing JSON data 
|   │   ├── screens - main screens of the application
|   │   ├── styles - Styled Components
|   │   ├── App.js
|   │   ├── index.js
|   │   ├── store.js (Redux)
│   ├── .gitignore
│   ├── package-lock.json
│   ├── package.json
├── README.md
```
